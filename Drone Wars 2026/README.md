# Drone Wars 2026

Run from the repo root:

- `npm install`
- `npm start`

That serves this folder at:

- `http://127.0.0.1:8080`

Notes:

- the game expects the `techtree/` and `assets/` folders to stay under `Drone Wars 2026/`
- the server is provided by the root `package.json` script, not by a separate package in this folder

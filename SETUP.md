# Setup

## Requirements

- Node.js 18 or newer

## Fresh Clone

From the repository root:

```bash
npm install
npm start
```

Open:

- `http://127.0.0.1:8080`

## Notes

- Drone Wars is served from the `Drone Wars 2026/` folder.
- Start it from the repository root, not from inside `Drone Wars 2026/`.
- The root `package.json` provides the `start` script.
